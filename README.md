# hot_news-email
# author: zhangwudi
# 2019/3/15 21:35:37

### 脚本说明
    定时发送邮件
    固定每天某个时间(比如早上6点)，根据sendEmail接口请求数据然后定时发送指定数据到指定邮箱

### 运行项目
 ## 安装
    pip3 install schedule
    pip3 install schedule
    pip3 install email
    pip3 install urllib
    pip3 install requests
    pip3 install
    pip3 install 

 ## 两种运行方式
    1.终端运行 python3 run.py
    2.运行项目根目录下的 run.py

### 脚本拓展
    同理：可以定时推送 自定义消息 、天气 、新闻等其他平台专属信息。。。
